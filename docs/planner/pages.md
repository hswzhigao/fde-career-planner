## 8. FDE 顾问对话 `/chat`

```text
┌─────────────────────────────────────────────────────┐
│  FDE 顾问对话                                        │
├──────────┬──────────────────────────────────────────┤
│ + 新对话 │  🎯 FDE 顾问对话                         │
│          │  向我提问任何关于 FDE 转型的问题          │
│ 会话1    │                                          │
│ 会话2    │  💬 我后端开发5年，转FDE需要学什么？     │
│ 会话3    │  💬 RAG 和 Agent 先学哪个？              │
│          │  💬 FDE 面试会问什么？                   │
│          │  💬 怎么做一个能打动面试官的作品集？     │
│          │                                          │
│          │  ┌─ 用户消息 ─────────────────────────┐ │
│          │  │ 我后端开发5年，转FDE需要学什么？    │ │
│          │  └────────────────────────────────────┘ │
│          │  ┌─ AI 回复（流式逐字显示）───────────┐ │
│          │  │ 好的，作为您的 FDE 顾问…▊          │ │
│          │  └────────────────────────────────────┘ │
│          │                                          │
│          │  [输入你的问题…__________] [发送]       │
└──────────┴──────────────────────────────────────────┘
```

交互：
- 左侧：会话列表，可新建/切换/删除
- 右侧：聊天区域，AI 回复流式逐字显示
- 推荐问题快捷入口（首次进入时）
- Enter 发送，Shift+Enter 换行
- 对话历史存 MySQL，刷新不丢
- 多轮上下文记忆，可追问

## 状态说明

| 状态 | 表现 |
|---|---|
| 空数据 | 显示引导文案 + 填写按钮 |
| 加载中 | skeleton 占位 |
| AI 调用中 | 按钮 loading + "AI 正在生成…" + 跳动指示器 |
| AI 流式输出 | 内容区域逐字显示 + 闪烁光标 ▊ |
| AI 完成 | "✓ 生成完成，已保存到数据库" |
| 错误 | 红色提示 + 重试按钮 |
| 对话加载中 | 三点跳动动画 |
| 对话流式输出 | 气泡内逐字显示 + 闪烁光标 ▊ |
